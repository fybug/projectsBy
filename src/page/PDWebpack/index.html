快速使用的手脚架

直接 clone master 分支即可使用。内置示例项目，位于 test 文件夹。

这是一个快速开发框架，或许你会觉得它不好用，因为它是设计用于开发其他基于 webpack 的手脚架时提供功能和结构支持。

更简洁的 webpack 使用

拆解庞大复杂的 webpack 配置文件到多个文件中，以原配置项名称命名，使用 IDE 的文件搜索功能更快捷的找到需要修改的配置。

使用自制的配置加载模块，在 run.js 中使用函数化的方式添加模块和页面，并在参数中将其连接起来。

被整合的插件

整合 HtmlWebpackPlugin 插件。

不再需要在插件中引入页面并手动连接所有模块。标记为公共模块的模块会自动加载到所有的页面中，可通过页面加载模块中传入页面的附加配置参数取消。

整合 ScriptExtHtmlWebpackPlugin 插件。

不在需要单独声明模块加载方式。在 js 模块加载模块中传入加载方式参数即可

整合 MiniCssExtractPlugin 插件。
